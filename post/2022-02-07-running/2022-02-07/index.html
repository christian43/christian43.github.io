Running is just controlling after fall

What? Why?

Today i was running my short 4km cycle. Sometimes while i am running i catch my self how i check and analyse my running technique.

Some thoughts today

  • keep an upright running position (make yourself big)
  • hold hands loosely with elbows flexed (triangle)
  • minimize braking forces during foot strike
  • spend some power in hopping (vertical force)
  • try to run light-footed

Just start running and do not change (intensity and volume) until you are able to run 45min with a stable heart rate pattern
