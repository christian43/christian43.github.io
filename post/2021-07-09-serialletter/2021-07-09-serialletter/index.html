Serial documents with LaTeX

July 9, 2021
LaTeX

Serial Documents

Often i have a task were i have to send documents or letters with an equal layout but different informations on it. The most prominient task surely are serial letters. To solve this task i use either MS Word or preferably LaTeX. The latter impresses with its excellent typesetting system.

What you need

  1. data source
  2. LaTeX template

The data source often comes as a plain formatted csv file. Sure MS Word also reads Excel files directly. LaTeX does not. Here we need a csv or text file.

Data source

Here i use a simple csv-file with the following structure. Columns are spearated by semicolon.

Name;Name1   
Fox;Joe   
Bunny;Bugs   
Duck;Duffy

LaTeX template

The LaTeX template may be defined as follows:

\documentclass[]{article}            
\usepackage[utf8]{inputenc}
\usepackage{datatool}
\DTLsetseparator{;}

\begin{document}

\DTLloadrawdb{test}{test.csv}
\DTLforeach{test}{\Name=Name, \lastname=Name1}{
	\textbf{\Name} \lastname \newpage
}

\end{document}

Summary

The output is written as a pdf document with three pages, were on each page the name is written. DTL iterate line by line through the database “test”. Now you are free to modify the LaTeX template with standard letter templates or your own template.
