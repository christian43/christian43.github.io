Image to Sketch

August 20, 2021
ImageMagick

Intro

Sometimes it is just nice to create a sketch from an image. There is from my opinion no other reason then fun on aesthetics.

Tool

There are plenty tools to solve such a task. Apps, browser based version or well known software tool like Gimp or Photoshop. Here i used ImageMagick a really powerfull image editing tool.

Code

At first we will need a picture. Every stands and falls with the picture. Not all images are suitable to create a sketch. Second a few lines of code.

convert image.jpg -intensity Rec709luminance -colorspace gray \
          \( +clone -blur 0x2 \) +swap -compose divide -composite \
		            -linear-stretch 5%x80%   image_sketch.png
