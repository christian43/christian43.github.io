Using R and Vim together

October 29, 2021
Vim R

Turn Vim into an R IDE

For some time I have been useing vim and R together and I am very happy with that solution. In this post I just want to describe how it works.

What is an IDE

IDE is the abbreviation for “integrated development environment”. To keep it simple IDE is a working environment to solve coding tasks. On Wikipedia you can find a full explanation. Why don’t I use R Studio? Hmmm, well I started to use vim and have gotten used to it.

What do you need?

  1. Command-line, Terminal or Shell
  2. Vim
  3. R
  4. Tmux
  5. vim-slime

What is vim-slime

By using vim-slime it is possible to send any text to another tmux pane (or even vims :terminal). Or even simplier:

Grab some text and send it to a target…

Visit the github repository from Jonathan Parlady and find out more about vim-slime.

Summary

This is a great plug-in for vim because you can use it for all other programming languages as well, e.g. Julia, Octave, Image Magick and much more.
